## Установка

- Node version: 20.17

```bash
npm i

```

## Структура папок

```
+-- src/

        |    +-- components/   # Все компоненты
            |   +-- ui/     # Компоненты ui
            |   +-- app/    # Цельные Компоненты приложения
        |    +-- core/      # Провайдеры, контекст
        |    +-- hooks/        # Хуки
        |    +-- layouts/      # Лайауты приложение
        |    +-- pages/        # Страницы
        |    +-- lib/          # Вспомогательные части (утилиты и т.д.)
        |    +-- typings/      # Декларация типов
        |   +-- assets/
            |    +-- fonts/                # Шрифты
            |    +-- img/                 # Картинки
            |    +-- icons/                # svg иконки
        |   +-- index.css                  # Cтили и подключение tailwind
        |   +-- App.tsx                    # Объявление роутов, инициализация miniApp
        |   +-- main.tsx                   # Провайдеры и рендер App
        |   +-- routes.tsx                 # Роуты приложения
```

## Запусе дев режима

```bash
npm run dev
```
